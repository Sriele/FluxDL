# FluxDL

Desktop application shell built with Tauri, React, and TypeScript.

## Development

Install dependencies:

```bash
pnpm install
```

Run the web frontend:

```bash
pnpm dev
```

Run the desktop app:

```bash
pnpm tauri:dev
```

Build the frontend:

```bash
pnpm build
```

Run Rust tests:

```bash
cd src-tauri
cargo test
```
